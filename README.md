# darktable-pixelpipe

A fork of darktable focused on extracting the image processing/pixelpipe components as a standalone library.

## Purpose

This project aims to isolate darktable's powerful image processing pipeline for use in other applications.

## License

GPL-3.0 (same as darktable)

## Key Directories

- src/develop/ - Core pixelpipe processing
- src/iop/ - 92 image operation modules
- src/common/ - Utilities and color management
- src/imageio/ - File format support
